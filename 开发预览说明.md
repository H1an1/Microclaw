# 预览说明

## 第一步：安装 Node.js（只需做一次）

按 Command + 空格，搜索"Terminal"并打开，然后把下面这行粘贴进去，按回车：

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

等它跑完之后，**关掉 Terminal 重新打开**，再粘贴：

```
nvm install --lts
```

等安装完成即可。

---

## 第二步：安装依赖（只需做一次）

打开 Terminal，粘贴以下内容按回车：

```
cd 
```

然后把 `desktop` 文件夹直接**拖入** Terminal 窗口（路径会自动填入），再按回车。

接着粘贴：

```
npm install
```

等待安装完成（可能需要 1-2 分钟）。

---

## 第三步：打开预览

在 Terminal 中粘贴以下内容，按回车：

```
npm run preview
```

等待约 5 秒，app 会自动打开。以后每次预览重复这一步即可。

---

## 什么时候用哪个命令

### 只预览前端页面 / 视觉效果

继续使用：

```
npm run preview
```

这个命令会重新构建 `renderer/dist`，然后替换到现有的 `.app` 里，适合看界面样式、图片、排版这些前端变化。

### 预览完整桌面端行为

如果你改了这些内容，请使用：

```
npm run preview:desktop
```

适用场景：

- 窗口大小变化
- 关闭 / 最小化 / 最大化
- 安装向导切换
- Electron 主进程逻辑
- IPC 通信

这个命令会先重新构建整个项目，再直接启动本地 Electron，所以能看到主进程改动。

---
